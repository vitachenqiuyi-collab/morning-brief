# 环球出行晨报

一个公开的静态晨报网站，固定为“头条 + 航班、签证、国际、其他”四张分类卡片。每次生成只会更新 `data/today.json`，不会改变网页结构。

## 本地查看

直接双击打开 `index.html` 即可查看版式。首次抓取资讯需要已安装 Node.js 18 或更高版本，然后在此文件夹执行：

```powershell
npm run generate
```

随后用任一静态服务器预览，例如：

```powershell
npx serve .
```

打开命令显示的本地地址；这样浏览器才能正常读取 `data/today.json`。

## 自动生成和公开发布（GitHub Pages）

1. 在 GitHub 新建一个公开仓库，将本文件夹全部上传。
2. 在仓库的 **Settings → Pages**，选择 **Deploy from a branch**，选择 `main` 分支与根目录 `/`。
3. 在 **Actions** 中允许工作流写入仓库内容（若组织策略有此限制）。
4. 工作流会在每天北京时间 08:15 运行；也可在 Actions 页面手动运行 `Generate daily morning brief`。

发布地址会显示在 GitHub Pages 设置页面，之后可绑定自己的域名。

## 信息源和规则

- 采集逻辑在 `scripts/generate-report.mjs`。直接读取中国民航局、国家移民管理局、外交部领事服务网、外交部重要新闻，以及澳大利亚、新西兰、挪威、冰岛相关官方政府部门与航空主管/机场的新闻或公告栏目；不使用泛新闻检索或搜索引擎结果。
- 只收录发布时间在最近 7 天内、且带可靠发布时间的资讯；同标题资讯会合并。约 28 小时内的资讯优先，其次按中国关联、签证、航班、政策变动等关键词计分排序。
- 每条资讯仅展示不超过 40 字的标题式一句话，方便快速阅读；点击可查看原始报道与正式公告。
- 各分类最多展示 4 条，首条全局最高分资讯作为头条。
- 这是资讯发现工具，不是官方出入境或航班公告。对行程有影响的信息，请始终打开原始链接并以航空公司、使领馆和政府官网公告为准。

要添加自己关注的来源或关键词，修改脚本顶部的 `feeds` 数组后重新运行生成命令即可。
